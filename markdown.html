What is Markdown?

It is a simple markup language used for formatting text in a plain-text editor.

Basic functionalities

  • put a word in bold
  • put a word in italic
  • strike a word

Creating Headers

We use the symbol "#" to mark a title or a subtitle. The most important header should have a single #, while the smaller ones have multiple "#" in a row. We can use up to six symbols for different heading sizes. Example:
